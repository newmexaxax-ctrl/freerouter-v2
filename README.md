# FreeRouter

Curated gateway to free AI tools. Minimal Next.js referral hub.

## Stack

Next.js · TypeScript · Tailwind · Motion · Lucide

## Develop

```bash
npm install
npm run dev
```

## Docs (Obsidian)

See `docs/` — start with [[Architecture]] in Obsidian, or open `docs/Architecture.md`.

## Scripts

| Command | Purpose |
|---------|---------|
| `npm run dev` | Dev server |
| `npm run build` | Production build |
| `npm run start` | Serve build |
| `npm run lint` | ESLint |
