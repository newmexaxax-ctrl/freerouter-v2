# Design System

#freerouter #design

## Philosophy

Less is more. White space is intentional. Premium, trustworthy, no link-farm energy.

## Color

| Role | Hex |
|------|-----|
| Background | `#FAFAFA` |
| Primary text | `#000000` |
| Accents | `#1A1A1A`, `#333333` |
| Borders | `black` at low opacity |

No colorful brand palette by default — high contrast black on off-white.

## Typography

- **Minimum body:** `text-lg` (18px)+
- Headings: bold, tight tracking, large scale
- Line height: relaxed
- One idea per section; no dense paragraphs

## Layout

- Massive padding (`py-28`+ on sections)
- Max content width ~ `max-w-6xl`
- No sidebars, widgets, dashboards, comparison tables
- Single primary CTA per moment

## Interaction

- `cursor-pointer` on all clickables
- Hover: border, shadow, or color shift — no layout jump
- Focus rings: black ring + off-white offset
